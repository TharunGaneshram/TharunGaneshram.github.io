Real Time Mimic Robotic Hand

Technologies: STM32, ESP32, TensorFlow, OpenCV, SolidWorks, Stepper Motors
Key Skills: PWM, UART, Real-Time Control, Motor Synchronization, Object Detection, CAD Design

This project features a robotic hand controlled by five stepper motors, each corresponding to a finger, with input from an object detection system powered by TensorFlow. The STM32 microcontroller handles the motor control, while the ESP32 manages Bluetooth communication with a laptop running the object detection model on OpenCV webcam input. CAD software was used to design the robotic hand components, which were then 3D printed. The project merges several technologies, from real-time motor control using PWM and UART communication, to machine learning for gesture detection and mimicry. This unique combination of mechanical design, embedded systems, and machine learning provided a comprehensive learning experience, emphasizing how multiple domains interact to create complex, responsive systems.

FreeRTOS Hyperloop Pod

Technologies: FreeRTOS, CMSIS-RTOS2, STM32, Raspberry Pi
Key Skills: RTOS Design, Task Management, Embedded Systems, Real-Time Control, CAN

As part of the Waterloo Hyperloop design team, I re-architected their entire firmware system using FreeRTOS and the CMSIS-RTOS2 API. Using FreeRTOS as an environment for managing multiple tasks concurrently on the STM32 microcontrollers, I implemented the RTOS for three critical modules: Sensor Module, Motor Controller Module, and Battery Management System (BMS). The Sensor Module handled data from pressure sensors, IMUs, and MUXes with dedicated threads and sent this data to the Raspberry Pi through CAN. The Motor Controller Module included FIFO management and tasks for processing motor commands, speed control, and emergency functions, with a CAN thread for data handling. The BMS module monitored battery metrics, utilized PWM for cell balancing, and polled thermistors for multi-fan thermal control. This project showcased my ability to integrate RTOS functionalities into complex embedded systems, enhancing real-time task management and system efficiency.

Omni-Directional Bluetooth Car

Technologies: ESP32, STM32, Bluetooth, SolidWorks
Key Skills: Real-Time Control, UART, PWM, Wireless Communication, CAD Design

The Omni-Directional Bluetooth Car project was designed to explore real-time vehicle control using Bluetooth communication. The car features one powered pivot wheel and three support pivot wheels, controlled using an ESP32 for Bluetooth communication and an STM32 microcontroller for vehicle control. Input data is transferred over UART, split into speed and direction values, and used to adjust PWM signals for dynamic motion control. This project gave me hands-on experience with wireless communication, real-time control in embedded systems, and the effective use of PWM for motor control. Additionally, this project allowed me to play around with SolidWorks as well as see the end-to-end hardware-software integration of a system that reacts to external commands in real-time.

5-Bar Pick and Place Mechanism

Technologies: Arduino, Botboarduino, Servo Motors, Serial Communication
Key Skills: Servo Control, Smooth Movement, Payload Manipulation, Embedded Systems

This project involves the development of a 5-bar pick-and-place mechanism controlled by a Botboarduino microcontroller. The system uses servo motors for axis and gripper control, enabling smooth and precise movement of payloads. The firmware developed incorporates direct manipulation of servos through digital pins, which communicate serially with an external controller. This project helped build my foundation in motor control in embedded systems, where precise timing and smooth transitions between positions are crucial. Writing firmware for servo control and implementing a basic serial communication protocol showed me how embedded systems handle real-world mechanical tasks, such as automated manufacturing or robotics.

4D Chess Game

Technologies: Java, C++, Object-Oriented Programming
Key Skills: Inheritance, Encapsulation, Algorithm Design, Polymorphism

This 4D Chess Game project was a venture into the complexities of multidimensional board games, implemented in both Java and C++. The project focused on creating a 4-dimensional chessboard, with individual chess pieces coded as separate classes, each inheriting from a base ChessPiece class. It helped me build skills in managing polymorphic behavior for chess piece movements and implementing game logic for checking and checkmating conditions. Working with complex multidimensional arrays and classes helped solidify my understanding of memory management and efficient data structures. This project showed me how object-oriented programming can be applied to manage complex logic in a structured and reusable way.
